# MÓDULO 13 - Mockups y UI

## Tarea
Como actividad de este módulo el Estudiante creará sus propios Mockups.

### Entregables
Mockup de lo que el alumno imagine como menús dentro del juego, esto implica:
- Un **menú principal**
- Un **menú de pausa**
- Un **menú de fin de juego** (ya sea que se haya perdido o ganado)

### Instrucciones
1. Crea los mockups de los menús faltantes
2. Exporta tus mockups a formato JPG
3. Comparte los mockups con tu tutor
4. No olvides actualizar tu repositorio con lo que vimos en este módulo

### Herramientas recomendadas
- [Diagrams.net](https://app.diagrams.net/)
- [NinjaMock](https://ninjamock.com/)
